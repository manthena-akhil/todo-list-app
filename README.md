📝 To-Do List App

A **responsive To-Do List web application** built using **HTML, CSS, and JavaScript** that allows users to add and delete tasks easily.
This project demonstrates **DOM manipulation, responsive design, and dynamic UI updates** using vanilla JavaScript.

---

🚀 Features

* ➕ Add new tasks
* 🗑 Delete tasks instantly
* 📱 Responsive design for different screen sizes
* 🎨 Clean and minimal UI
* 📜 Scrollable task list with custom scrollbar
* ⚡ Built using pure JavaScript (no frameworks)

---

🛠 Technologies Used

* HTML5 – Structure of the application
* CSS3 – Styling and responsive layout
* JavaScript (ES6) – DOM manipulation and functionality

---

 📂 Project Structure

```
todo-list-app
│
├── index.html   # Main HTML file
├── index.css    # Styling for the app
└── script.js    # JavaScript functionality
```

---

 💻 How to Run the Project

1. Clone the repository

```
git clone https://github.com/manthena-akhil/todo-list-app.git
```

2. Open the project folder

```
cd todo-list-app
```

3. Open **index.html** in your browser.
